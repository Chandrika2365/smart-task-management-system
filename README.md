# Smart Task Management System

## Features
- Add Tasks
- Update Tasks
- Delete Tasks
- Analytics Dashboard
- Completion Percentage

## Technologies Used
- Python
- Flask
- SQLite
- Pandas
- NumPy
- HTML
- CSS

## Setup Steps

### Install Packages

```bash
pip install flask flask_sqlalchemy pandas numpy
```

### Run Project

```bash
python app.py
```

### Open Browser

```text
http://127.0.0.1:5000
```

## Project Structure

- app.py
- templates/
- static/
- taskdb.db
